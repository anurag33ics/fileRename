# File Rename Utility

This repository is designed for reading files from folders and generating an Excel sheet that organizes the data. The resulting Excel file provides details about the files, including a new file number for each.

## Features

The generated Excel file will include the following columns:
1. **Sr No**: Serial number for each file.
2. **Folder Name**: The name of the folder containing the file.
3. **Original File Name**: The file's original name.
4. **New File Number**: A new number assigned to the file.

## Prerequisites

Ensure you have [Node.js](https://nodejs.org) installed on your machine.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anurag33ics/fileRename.git
   ```
2. Navigate to the project directory:
   ```bash
   cd fileRename
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage

Run the project using the following command:

```bash
node index.js
```
